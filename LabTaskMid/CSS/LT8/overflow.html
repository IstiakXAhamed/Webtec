Overflow Handling Demo

overflow: visible

This is a long block of text designed to test overflow behavior. It will extend outside the box when the content is larger than the height. This helps demonstrate how the visible overflow mode works. More text is added so that it clearly exceeds the 200px height and you can see the effect.

overflow: hidden

This long text will demonstrate how overflow hidden clips the extra content. Anything that does not fit inside the 200px box will be cut off cleanly without any scrollbars. This shows how hidden controls the visible content area.

overflow: scroll

Scroll will always show scrollbars even if content barely exceeds the container. This allows the user to scroll through all hidden text. This paragraph contains enough text to force scrolling and display the scrollbar inside the box for easy testing of overflow behavior.

overflow: auto

Auto only shows scrollbars when needed. If the content fits, no scrollbars appear. Otherwise, scrollbars show automatically. This is the most commonly used mode for handling dynamic text inside a fixed container.
